MORE IMAGES PAGE

This package contains:

  more-images.html        The complete page
  more-images/assets/     The renamed, web-sized images, GIF, video, and texture

To add it to your GitHub Pages site:

1. Put more-images.html beside index.html and photos.html.
2. Put the entire more-images folder beside it. Do not put the folder inside
   another folder; the page uses those exact relative paths.
3. Add this link wherever you want the new page to appear:

   <a href="more-images.html">More Images</a>

The page is self-contained and does not use outside fonts, scripts, trackers,
or services. The duplicate copy of the abstract emblem was intentionally left
out. The sideways Mark Twain notice was rotated and converted into the slow
oxblood-red GIF.
